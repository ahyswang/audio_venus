# AUDIO_VENUS

通用的项目框架

## 环境部署

* 1.安装python运行环境。
```
conda create -n audio_venus python=3.7.3
conda activate audio_venus
pip install -r requirements.txt
```

## 源码编译 

* 1.linux64编译。
```
 ./build_linux.sh
```
